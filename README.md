## Requirements

- Xcode 10 or later
- iOS 12


## How to build

- Change the **Team** setting in the **General** tab for each target.
- Build & Run with **Xcode 10** or later

## How to use

- Launch iOS app with Xcode
- Kill app in app switcher
- Launch iOS app again with Xcode
